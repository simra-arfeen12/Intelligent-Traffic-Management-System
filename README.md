# Intelligent Traffic Management System

## Project Overview

The Intelligent Traffic Management System is an Artificial Intelligence and Computer Vision based project that monitors traffic using the YOLOv8 object detection model. The system detects and classifies vehicles, estimates traffic density, and dynamically adjusts traffic signal timings based on traffic conditions.

## Features

- Vehicle Detection using YOLOv8
- Vehicle Classification (Cars, Motorcycles, Buses, Trucks)
- Vehicle Counting
- Traffic Density Analysis
- Dynamic Traffic Signal Timing
- Illegal Parking Detection (Prototype)
- Accident Detection (Prototype)
- Incident Alert System (Prototype)
- Traffic Operations Dashboard
- Vehicle Distribution Analysis using Bar Chart and Pie Chart
